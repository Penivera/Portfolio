Open Source / Rust + WASM

IronClaw (NEAR AI)

I added GitHub and Discord WASM components to IronClaw, NEAR AI's agentic OS. The PR added 1,756 lines across 8 files and went through 53 review comments before being merged. The challenge was implementing capability-based security so plugins could interact with external APIs without compromising the host system.
